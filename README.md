# Portafolio Web - Davith Morales

Este es el repositorio de mi portafolio web, donde muestro mi perfil profesional, mis habilidades técnicas y proyectos. Este sitio está desarrollado con **HTML**, **CSS**, y **JavaScript**, utilizando módulos y estilos personalizados para crear una experiencia atractiva y funcional.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- `index.html`: Archivo principal que contiene la estructura del sitio web.
- `estilo.css`: Archivo de estilos que define el diseño visual de la página.
- `script.js`: Archivo JavaScript que controla la interactividad de la página, como la navegación y la animación de la barra de habilidades.
- `img/`: Carpeta que contiene imágenes utilizadas en el portafolio (logotipos, fotos, etc.).
- `Curriculum Davith Morales.pdf`: Archivo PDF con mi currículum disponible para descarga.

## Secciones del Portafolio

- **Inicio**: Una presentación con mi nombre, profesión, y enlaces a redes sociales.
- **Sobre mí**: Información personal como mi ubicación, intereses, y contacto.
- **Habilidades**: Gráficas que muestran mis habilidades técnicas y profesionales.
- **Currículum**: Mi trayectoria académica y profesional en detalle.
- **Portafolio**: Galería de proyectos web y otros trabajos visuales.
- **Contacto**: Formulario para que los visitantes puedan contactarme directamente.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura semántica del contenido.
- **CSS3**: Para el diseño y la presentación visual.
- **JavaScript**: Para la interactividad y funcionalidad de la página.
- **Font Awesome**: Para los íconos en la barra de navegación y las secciones de habilidades.
- **Responsive Design**: El diseño es adaptable a dispositivos móviles y de escritorio.

## Características

- Diseño **responsive** para una visualización óptima en todos los dispositivos.
- **Formulario de contacto** validado con JavaScript y HTML5.
- **Animación** de las barras de habilidades mediante CSS y JavaScript.
